명령어가 매개변수 dirs로 주어질 때, 게임 캐릭터가 처음 걸어본 길의 길이를 구하여 return 하는 solution 함수를 완성해 주세요.

##### 제한사항

- dirs는 string형으로 주어지며, 'U', 'D', 'R', 'L' 이외에 문자는 주어지지 않습니다.
- dirs의 길이는 500 이하의 자연수입니다.

##### 입출력 예

| dirs        | answer |
| ----------- | ------ |
| "ULURRDLLU" | 7      |
| "LULLLLLLU" | 7      |

- 문제 풀이

```javascript
// 첫번째 풀이
function solution(dirs) {
    let now = [0,0]
    let arr = new Set()
    for(let i=0; i<dirs.length; i++){
        let x=now[0],y=now[1]
        if(dirs[i] === 'U') y=now[1] + 1
        if(dirs[i] === 'D') y=now[1] - 1
        if(dirs[i] === 'R') x=now[0] + 1
        if(dirs[i] === 'L') x=now[0] - 1
        if(x > 5 || x<-5 || y>5 || y<-5) continue;
        arr.add(""+now[0]+now[1]+x+y)
        arr.add(""+x+y+now[0]+now[1])
        now = [x,y]
    }
    return arr.size/2;
}

- 질문하기와 풀이에서 힌트를 얻어 풀이 진행
- 현재 좌표를 나타내는 now=[0,0] 설정, arr을 set으로 설정
- for문을 통해 dirs 문자열을 돌면서
- x=now[0], y=now[0]으로 먼저 설정하고, dirs[i]가 무엇인지에 따라 x,y값 변경
- 여기서 x,y값은 now좌표에서 이동할 좌표
- x,y가 5보다 크거나 -5보다 작으면 continue
- arr에 "now[0]now[1]xy"값과 "xynow[0]now[1]"의 값을 add
- ex. [0,0] -> [1,0]이라면, 이후에 [0,0]->[1,0]과 [1,0]->[0,0]은 이미 지나간 길로 체크되어야 하기 때문
- add가 끝나면 now = [x,y]로 바뀐 좌표로 설정
- set은 중복을 허용하지 않기 때문에 이미 지나간 길은 체크되지 않고, 왕복거리를 계산한 것이기 때문에 arr.size/2를 해주면 정답
```
